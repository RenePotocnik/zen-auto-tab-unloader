# Zen Tab Unloader

A lightweight extension designed specifically for [Zen Browser](https://zen-browser.app/) to handle tab unloading across Workspaces.

## Features
- **Workspace Aware:** Unloads hidden tabs in other workspaces.
- **Essential vs Pinned:** Configurable logic to distinguish between Global Essentials and Workspace Pins.
- **Audio Protection:** Prevents unloading tabs playing/using audio.
